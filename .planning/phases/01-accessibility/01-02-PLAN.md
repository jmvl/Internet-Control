# 01-02: Visible Focus Indicators

<objective>
Add comprehensive focus indicators to all interactive elements using :focus-visible, ensuring WCAG AA compliance and clear keyboard navigation feedback.
</objective>

<context>
@websites/www.dinero.cash/assets/css/accessibility-improvements.css
@websites/www.dinero.cash/index.html
@websites/www.dinero.cash/users.html
@websites/www.dinero.cash/agents.html
@.planning/STATE.md
</context>

<tasks>
<task name="Create focus-visible CSS rules">
1. Open `assets/css/accessibility-improvements.css`
2. Add at top of file:
   ```css
   /* Focus indicators for keyboard navigation */
   :focus-visible {
     outline: 3px solid #4d90fe;
     outline-offset: 2px;
   }

   /* Remove outline for mouse-only interactions */
   :focus:not(:focus-visible) {
     outline: none;
   }

   /* Ensure all interactive elements can receive focus */
   a, button, .button, .nav-link, .feature-box, input, textarea, select {
     focus-visible: auto;
   }
   ```
3. Save file
</task>

<task name="Apply focus styles to navigation">
1. Open accessibility-improvements.css
2. Add specific focus styles for navigation:
   ```css
   .nav-link:focus-visible {
     outline: 3px solid #4d90fe;
     outline-offset: 4px;
     border-radius: 4px;
   }

   .menu-button:focus-visible {
     outline: 3px solid #4d90fe;
     outline-offset: 2px;
   }
   ```
3. Save file
</task>

<task name="Apply focus styles to buttons and CTAs">
1. Open accessibility-improvements.css
2. Add focus styles for buttons:
   ```css
   .button:focus-visible,
   button:focus-visible {
     outline: 3px solid #4d90fe;
     outline-offset: 3px;
     box-shadow: 0 0 0 2px rgba(77, 144, 254, 0.3);
   }
   ```
3. Save file
</task>

<task name="Apply focus styles to interactive elements">
1. Open accessibility-improvements.css
2. Add focus styles for feature boxes and form elements:
   ```css
   .feature-box:focus-visible {
     outline: 3px solid #4d90fe;
     outline-offset: 4px;
   }

   input:focus-visible,
   textarea:focus-visible,
   select:focus-visible {
     outline: 3px solid #4d90fe;
     outline-offset: 2px;
   }
   ```
3. Save file
4. Test in browser by tabbing through all elements
</task>
</tasks>

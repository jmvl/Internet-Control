# 01-06: Image Alt Text Completion

<objective>
Add descriptive alt text to all informative images and `alt=""` to decorative images, ensuring screen reader users have equivalent access to visual content.
</objective>

<context>
@websites/www.dinero.cash/index.html
@websites/www.dinero.cash/users.html
@websites/www.dinero.cash/agents.html
@.planning/STATE.md
</context>

<tasks>
<task name="Audit all images for missing or empty alt text">
1. Open `index.html`
2. Search for all `<img>` tags (use grep: `grep -n '<img' index.html`)
3. For each image, check if alt attribute is:
   - Missing entirely
   - Empty (`alt=""`) when image is informative
   - Non-descriptive (`alt=""` or `alt="image"`)
4. Create list of images needing alt text
5. Repeat for users.html and agents.html
6. Save audit list in memory for reference
</task>

<task name="Add alt text to hero images in index.html">
1. Open `index.html`
2. Find hero images (search for `hero-1.jpg`, `hero-2.jpg`, etc.)
3. For each hero image, write descriptive alt text:
   - Hero 1: `alt="Mobile wallet app interface showing banking and payment features on smartphone screen"`
   - Hero 2: `alt="Business growth dashboard showing user engagement analytics and customer base expansion"`
   - Hero 3: `alt="Person using smartphone for digital banking and mobile payments at retail store"`
   - Hero 4: `alt="Customizable business platform dashboard showing modular payment ecosystem configuration"`
4. Save index.html
</task>

<task name="Add alt text to tab interface images">
1. Open `index.html`
2. Find tab images (`tabs-users.png`, `tabs-business.png`, `tabs-banks.png`)
3. Add descriptive alt text:
   - tabs-users.png: `alt="Dinero Cash mobile app interface showing user payment features including buy/sell goods and ecosystem services"`
   - tabs-business.png: `alt="Dinero Cash merchant dashboard showing agent network management, cash operations, and payment tools"`
   - tabs-banks.png: `alt="Dinero Cash banking platform interface for cross-border operations and multi-country management"`
4. Save index.html
</task>

<task name="Add alt text to feature card images">
1. Open `index.html`
2. Find all feature card images in slider (wallet-*.png, payments-*.png, merchant-*.png, onboarding-*.png)
3. For each image, write alt text describing the specific feature shown:
   - wallet-1.png: `alt="Money transfer interface showing send money screen"`
   - wallet-2.png: `alt="Multiple wallet display on single screen"`
   - wallet-3.png: `alt="Contact list showing people to pay from wallet"`
   - [Continue for all 24 feature card images]
4. Use pattern: `alt="[Feature name] interface showing [description of what's visible]"`
5. Save index.html
</task>

<task name="Add alt text to users.html and agents.html images">
1. Open `users.html`
2. Find all `<img>` tags
3. Add appropriate alt text following same pattern as index.html
4. Open `agents.html`
5. Find all `<img>` tags
6. Add appropriate alt text
7. Save both files
</task>

<task name="Verify alt text with screen reader">
1. Open index.html in browser
2. Enable screen reader (VoiceOver on macOS: Cmd+F5)
3. Navigate through images
4. Verify alt text is read correctly and provides meaningful description
5. Check for any images that should be decorative (add `alt=""`)
6. Repeat for users.html and agents.html
</task>
</tasks>

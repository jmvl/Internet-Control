# 01-01: Skip Navigation Link

<objective>
Add WCAG-compliant skip navigation link to all pages, allowing keyboard users to bypass navigation and jump directly to main content.
</objective>

<context>
@websites/www.dinero.cash/index.html
@websites/www.dinero.cash/users.html
@websites/www.dinero.cash/agents.html
@websites/www.dinero.cash/assets/css/accessibility-improvements.css
@.planning/STATE.md
</context>

<tasks>
<task name="Create skip-link CSS class">
1. Open `assets/css/accessibility-improvements.css`
2. Add `.skip-link` class with:
   - `position: absolute; top: -100px; left: 0` (hidden off-screen)
   - `background: #000; color: #fff; padding: 12px 24px` (high contrast)
   - `z-index: 9999` (above all content)
   - `transition: top 0.3s` (smooth reveal)
3. Add `:focus` state with `top: 0` to reveal when focused
4. Save file
</task>

<task name="Add skip link HTML to all pages">
1. Open each HTML file: index.html, users.html, agents.html
2. Find `<body>` tag (line ~2 in each file)
3. Immediately after `<body>`, add: `<a href="#main-content" class="skip-link">Skip to main content</a>`
4. Find main content wrapper (has class `page` or `hero` section)
5. Add `id="main-content"` to the main content wrapper
6. Save all files
</task>

<task name="Verify skip link functionality">
1. Open index.html in browser
2. Press Tab key - skip link should become visible
3. Press Enter - page should scroll to main content
4. Repeat for users.html and agents.html
5. Confirm functionality works on all pages
</task>
</tasks>

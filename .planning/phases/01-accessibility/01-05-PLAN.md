# 01-05: Color Contrast Audit & Fixes

<objective>
Audit all color combinations for WCAG AA compliance (4.5:1 for normal text, 3:1 for large text) and fix any failing combinations.
</objective>

<context>
@websites/www.dinero.cash/assets/css/dinerocash.webflow.shared.f0a36ca9c.css
@websites/www.dinero.cash/index.html
@websites/www.dinero.cash/users.html
@websites/www.dinero.cash/agents.html
@https://webaim.org/resources/contrastchecker/
@.planning/STATE.md
</context>

<tasks>
<task name="Document current color palette">
1. Open `dinerocash.webflow.shared.f0a36ca9c.css`
2. Extract all color values used for:
   - Text colors (`.p`, headings, body text)
   - Background colors
   - Button colors and states
   - Link colors
   - Border colors
3. Create markdown document `websites/www.dinero.cash/docs/color-contrast-audit.md`
4. List each color combination with foreground/background hex values
5. Save document
</task>

<task name="Audit color combinations with WebAIM checker">
1. Visit https://webaim.org/resources/contrastchecker/
2. For each documented color combination:
   - Enter foreground hex value
   - Enter background hex value
   - Record contrast ratio
   - Note if it passes WCAG AA (normal text: 4.5:1, large text: 3:1)
3. Focus on high-risk combinations:
   - Hero text on image backgrounds
   - Grey text on light backgrounds (`.p`, `._14`)
   - Button text on colored backgrounds
   - Links in context
   - Form field borders
4. Update color-contrast-audit.md with results
5. Mark each combination as PASS/FAIL with ratio
</task>

<task name="Fix failing color combinations">
1. For each FAIL in audit, determine fix strategy:
   - Adjust foreground hex value to lighter/darker
   - Add semi-transparent background overlay
   - Add text-shadow for contrast
2. Apply fixes in `accessibility-improvements.css`:
   ```css
   /* Fix: Low contrast grey text */
   .p._14 {
     color: #424242; /* Changed from #757575 for better contrast */
   }

   /* Fix: Text on image backgrounds */
   .hero-content {
     text-shadow: 0 2px 8px rgba(0,0,0,0.5);
   }

   /* Fix: Light text on light background */
   .button.light {
     color: #212121; /* Changed from #757575 */
   }
   ```
3. Re-check fixed combinations with WebAIM
4. Update color-contrast-audit.md with fix details
5. Save CSS file
</task>

<task name="Test with color blindness simulator">
1. Visit https://www.toptal.com/designers/colorfilter
2. Enter dinero.cash URL or load local HTML
3. Test with each simulation:
   - Protanopia (red-blind)
   - Deuteranopia (green-blind)
   - Tritanopia (blue-blind)
   - Achromatopsia (monochromacy)
4. Document any readability issues
5. If issues found, adjust colors and re-test
6. Update color-contrast-audit.md with findings
</task>
</tasks>
